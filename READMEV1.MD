Este projeto representa o estágio inicial de um sistema de gestão de vendas, focado na experiência do usuário (UX) e validação de requisitos fundamentais de front-end.

✨ Funcionalidades
Catálogo Dinâmico: Listagem de produtos através de manipulação de arrays em JavaScript.

Calculadora de Pedidos: Lógica para cálculo de subtotais e descontos aplicada diretamente no DOM.

Busca de Endereço: Integração via fetch com a Brasil API para preenchimento automático de dados através do CEP.

Design Responsivo: Construído com Tailwind CSS para adaptação em dispositivos móveis.

🛠️ Tecnologias
HTML5 / CSS3 (Tailwind)

JavaScript Vanilla (ES6)

Brasil API (CEP)
