VendaSync - Sistema de Gestão de Vendas (Parte 1)Este projeto é o estágio inicial de um sistema de gestão voltado para pequenos comerciantes, focado em agilizar o registro de pedidos e a validação de endereços de entrega.📁 Estrutura do ProjetoO projeto foi desenvolvido em uma estrutura de arquivo único para facilitar a portabilidade e visualização imediata:HTML5: Estrutura semântica dos componentes.Tailwind CSS: Framework utilitário para design moderno e responsivo.JavaScript (Vanilla): Lógica de manipulação do DOM, cálculos e consumo de API.🚀 Como Rodar o SistemaLocalmenteCertifique-se de ter um navegador moderno instalado (Chrome, Edge ou Firefox).Salve o código fornecido no arquivo index.html.Abra o arquivo diretamente no seu navegador clicando duas vezes sobre ele.Via GitHub (Live Demo)No seu repositório, acesse Settings > Pages.Selecione a branch principal e clique em Save.Importante: Para que o link funcione automaticamente, o arquivo principal deve se chamar index.html.Link do Projeto: https://jacianareginadasilva.github.io/-VendaSync/Nota: Para a integração do CEP funcionar, é necessária uma conexão ativa com a internet.✨ Funcionalidades ImplementadasCatálogo de Produtos: Armazenado em um array de objetos JavaScript, exibido dinamicamente na interface.Calculadora de Descontos: Aplicação de porcentagem em tempo real sobre o produto selecionado.Consumo de API (Brasil API): Busca assíncrona de endereço a partir do CEP.Interface Responsiva: Design adaptável para mobile e desktop.
